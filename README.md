# UD6 - Datos Avanzados en Java

## Estructura del proyecto

src/net/salesianos/
 ├─ ejercicio1 → Reparaciones (Queue y Stack)
 ├─ ejercicio2 → Asistentes (Set y Map)
 └─ ejercicio3 → Videojuegos (CSV y ficheros)

## Ejercicio 1
Se gestiona una cola de reparaciones y un historial con una pila.

## Ejercicio 2
Se guardan asistentes sin duplicados usando Set y se cuentan por grupo con Map.

## Ejercicio 3
Se lee un CSV de videojuegos, se calculan datos y se generan dos ficheros:
- resumen_videojuegos.txt
- log_videojuegos.txt
